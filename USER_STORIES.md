# Habit Tracker User Stories

## Authentication 
### Account Registration
- **Story:** As a user, I want to register with my name, username, age, and country so that I can create an account and access the habit tracking features.
- **Acceptance Criteria:** - Fields for Name, Username, Age, and Country are present.
    - System validates that all fields are filled.
- **Priority:** High

### Account Login
- **Story:** As a user, I want to log in using my username and password so that I can access my account and track my habits.
- **Acceptance Criteria:**
    - User can enter a default username/password to enter the app.
- **Priority:** High

### Error Feedback
- **Story:** As a user, I want to receive a message if I enter the wrong username or password so that I know my login attempt was unsuccessful.
- **Priority:** Medium

---

## Homepage 
### View Welcome Message
- **Story:** As a user, I want to see a personalized welcome message with my name on the homepage, so that I feel recognized.
- **Acceptance Criteria:**
    - Header displays "Welcome, [User Name]".
- **Priority:** High

### Display Progress
- **Story:** As a user, I want to see my daily progress for each habit on the homepage, so that I can easily monitor my status.
- **Priority:** High

---

## Menu & Navigation 
### Access Menu Options
- **Story:** As a user, I want to access a menu with options for configuring habits, reports, profile, and signing out.
- **Acceptance Criteria:**
    - Hamburger icon opens a side drawer.
- **Priority:** High

### Sign Out
- **Story:** As a user, I want to sign out of my account using an option in the menu, so that I can securely log out.
- **Priority:** Medium

---

## Profile Page 
### Edit Personal Information
- **Story:** As a user, I want to update my name, username, age, and country on my profile page, so that I can keep my information up to date.
- **Acceptance Criteria:**
    - Updated name reflects immediately in the app header.
- **Priority:** Medium

---

## Habits Management 
### Add/Delete Habits
- **Story:** As a user, I want to add new habits or delete existing ones so that I can manage my routine.
- **Priority:** High

### Personalize with Color
- **Story:** As a user, I want to assign a specific color to each habit to make it visually distinct and personal.
- **Priority:** Low

---

## Reports Page
### View Weekly Reports
- **Story:** As a user, I want to see a report of my weekly habit progress so that I can understand my consistency.
- **Acceptance Criteria:**
    - Visual chart showing completed vs. incomplete habits.
- **Priority:** Medium

---

## Notifications 
### Manage Reminders
- **Story:** As a user, I want to enable/disable notifications and set specific times (morning, afternoon, evening) for habit reminders.
- **Priority:** Low
